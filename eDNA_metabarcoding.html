eDNA metabarcoding

El metabarcoding es una herramienta de gran poder para explorar la diversidad genética de diferentes sistemas biológicos. Este flujo de trabajo está diseñado para ser empleado sobre datos generados a partir de muestras del medio ambiente (suelo, sedimento, agua e incluso heces)

Apartir de datos de secuenciación masiva generados por la plataforma de Illumina, se realiza un ensamble de las lecturas pareadas para reconstruir al marcador seleccionado, posteriormente, aquellos ensambles que resultan muy cortos o muy largos son removidos. Después se realiza un proceso de dereplicación de secuencias, de filtrado de secuencias únicas y aquellas que presentan errores de PCR/secuenciación. Finalmente, empleando una base de datos previamente personalizada para el marcador se realiza la asignación taxonómica con una combinación de métodos que incluyen BLAST/LCA (Lowest Common Ancestor)

Flujo de trabajo:

Este proyecto no incluye:

  • Discusión ni interpretación de los resultados obtenidos

Entregables:

  • Resultados de control de calidad de secuencias crudas y procesadas (html)
  • Archivo con secuencias anotadas en formato fasta y CSV
  • Gráficos adaptados a las necesidades de cada proyecto
